# Blockchain-based-Messaging-dApp
Ethereum based Blockchain messaging Dapp

### Useful Commands

Reset cc
```
truffle migrate --reset
```
opening truffle cmdp from console
```
truffle console
```
running cc from truffle console
```
Message.deployed().then(function(i) {app =i; })
```
view accounts
```
app.web3.accounts
```
running server
```
npm run dev
```
